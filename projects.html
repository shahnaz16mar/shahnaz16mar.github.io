My Data Science Projects

Healthcare Classification Project

This project uses machine learning techniques, including Decision Trees, Logistic Regression, and Random Forests, to predict the presence of cardiovascular disease. The model is trained on various features, such as age, cholesterol levels, and blood pressure, to make accurate predictions. Scikit-learn is used for model development, while performance evaluation is conducted using standard metrics like precision, recall, and feature importance to ensure reliability and interpretability.

Anomaly Detection

This project uses XGBoost, Logistic Regression, and Decision Trees to detect fraudulent credit card transactions in a highly imbalanced dataset. Since minimizing false negatives is the top priority to prevent financial loss, the models are trained to maximize fraud detection. Scikit-learn and XGBoost are used for development, with performance evaluation focusing on recall and overall reliability to ensure fraudulent transactions are correctly identified.
